# Pong-Reseau
Pong game network


port server de demande de connection server: 5000

port server à l'écoute du client 1: 5010
port server à l'écoute du client 2: 5020

port client réception à l'écoute du serveur: 5001



Client: 
-  State 1: Menu de connexion (enter l'ip)
-  State 2: Menu d'attente de connexion au serveur (le client envoie "he oh" et attends le numero de portTx)
-  State 3: Menu d'attente de connexion du deuxième joueur
-  State 4: Jeu
